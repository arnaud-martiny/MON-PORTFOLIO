Ô ZEN Esthétique

Création d'une plateforme de réservation sur mesure

Le Contexte & Le Défi

L'institut Ô ZEN Esthétique dépendait historiquement de la plateforme TreatWell pour ses réservations. Le problème ? Des commissions exorbitantes réduisant drastiquement les marges, un manque de contrôle total sur le contenu, et une interface rigide. La cliente possédait un ancien site Wix inachevé et mal référencé. Le défi était de taille : créer une plateforme indépendante complète, capable de gérer les réservations, les paiements et le marketing, le tout avec un référencement local irréprochable.

Ma Solution Technique & Stratégique

J'ai pris la décision d'abandonner les CMS traditionnels pour développer une solution 100% sur mesure en PHP (plus de 103 fichiers et 18 000 lignes de code).

Front-End & Expérience Client : J'ai conçu un tunnel de réservation sans friction sous forme de "Single Page Application" (SPA) en Vanilla JS. Les clients peuvent composer leur séance, choisir leurs créneaux, et payer en ligne. Pour réduire les abandons, j'ai intégré un mode "Invité" (sans inscription obligatoire).

Paiement Sécurisé : Intégration native de l'API Stripe (sans plugins lourds) avec gestion asynchrone des Webhooks pour valider les créneaux en base de données.

Back-Office (Admin) : Création d'un panel d'administration ultra-sécurisé (Authentification 2FA, anti-brute force). Ce dashboard génère les créneaux horaires, gère le CRM client, et permet la création de codes promotionnels.

Acquisition & Tracking : Développement d'un système de newsletter interne avec intégration de pixels invisibles pour traquer les taux d'ouverture et de clics sans dépendre de solutions tierces comme Mailchimp.

Les Résultats

La cliente a pu s'affranchir des plateformes tierces. Le site affiche des scores "Core Web Vitals" de 100% (PageSpeed Insights), garantissant un SEO local puissant. L'interface est rapide (<800ms sur mobile), sécurisée (chiffrement AES-256 pour les données RGPD), et évolue chaque semaine selon les besoins métiers.

Stack Technique
PHP 8 PDO/MySQL Vanilla JS (ES6+) API Stripe API Brevo (SMTP) Architecture MVC Custom

Snoussi Studio

Vitrine B2B Premium & Code Pur

Le Contexte & Le Défi

Sofiane, gestionnaire de réseaux sociaux et créateur de contenu, avait besoin d'un site web pour asseoir la crédibilité de son studio (Photo, Branding, Podcast). L'objectif n'était pas de faire un simple site vitrine, mais une plateforme haut de gamme, ultra-rapide, capable de convertir des visiteurs B2B en clients réguliers, tout en se positionnant agressivement sur les moteurs de recherche.

Ma Solution Technique & Stratégique

Plutôt que d'utiliser Webflow ou WordPress, j'ai opté pour une approche "Back-to-Basics / Code Pur" afin de garantir une performance absolue.

Architecture & Vitesse : Site codé en PHP et HTML/CSS natif avec une base de données SQLite légère pour le système de réservation du studio. Résultat : un temps de chargement inférieur à 1.2s.

Immersion Visuelle (GSAP) : Pour l'effet "Waouh", j'ai intégré la librairie GSAP. Le site dispose d'animations fluides au scroll (ScrollTrigger), d'effets magnétiques sur les boutons, de distorsions au défilement (Skew on Scroll) et d'un curseur personnalisé morphologique.

Stratégie SEO : Création d'une arborescence de plus de 15 pages de services ciblées (ex: "Photographe portrait Bruxelles"). J'ai implémenté de l'URL Rewriting propre (.htaccess) et injecté des microdonnées JSON-LD (FAQPage, LocalBusiness) pour dominer les résultats de recherche Google.

Sécurité anti-spam : Plutôt que d'utiliser un reCAPTCHA frustrant, j'ai codé un système de "Honeypot" invisible qui piège les robots, couplé à une interface d'administration sécurisée par jeton OTP (2FA par email).

Les Résultats

Le site est un véritable aimant à prospects. L'expérience utilisateur premium alliée au référencement chirurgical a permis au studio de signer de multiples nouveaux contrats B2B peu après le lancement.

Stack Technique
PHP Custom SQLite 3 JavaScript (Vanilla) GSAP (GreenSock) CSS3 natif

Le Théâtre des Poètes ASBL

Résurrection Digitale

Le Contexte & Le Défi

Cette ASBL culturelle se trouvait dans une situation d'urgence : leur site web, datant de 2015, avait été piraté. Il était non-responsive, visuellement obsolète, et totalement impossible à mettre à jour pour l'équipe (qui n'avait aucune compétence technique). Le budget était limité, mais le besoin d'une vitrine propre pour la rentrée théâtrale était critique.

Ma Solution Technique & Stratégique

Plutôt que de "rustiner" l'ancien site, j'ai pris la décision de tout raser pour reconstruire une architecture saine, sécurisée et sur mesure.

Base de Données & Sécurité : Conception d'une nouvelle structure MySQL propre. J'ai implémenté un système de requêtes préparées (PDO) pour éradiquer tout risque futur d'injection SQL.

Back-Office Ultra-Simplifié : Conscient que les utilisateurs finaux n'étaient pas des développeurs, j'ai créé un panel d'administration en PHP sur mesure, allant à l'essentiel : ajouter une pièce de théâtre, modifier une date, publier une actualité. Zéro superflu.

Front-End & Accessibilité : Développement d'une interface responsive, respectant les normes d'accessibilité web (contraste, balisage sémantique) pour mettre en valeur les visuels de la troupe et faciliter la lecture du programme.

Les Résultats

Un lancement réussi pour la nouvelle saison. L'ASBL a retrouvé son indépendance digitale avec un outil rapide, immunisé contre les failles courantes, et géré de manière autonome par l'équipe administrative.
